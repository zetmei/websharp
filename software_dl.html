    • AQUOS液晶電視Z5T系列 SKYPE視訊支援軟體更新
    • LC-60Z5T / LC-52Z5T
      LC-46Z5T / LC-40Z5T
    • 2015-01-20
    • 驅動程式
    • 下載

    一、說明：

    液晶電視 Z5T 系列 Skype視訊服務即日起提供軟體更新服務若您的電視為 2011年9月以後生產的已經內建此軟體，不需更新。

    二、適用機種：

    LC-60Z5T、LC-52Z5T、LC-46Z5T、LC-40Z5T

    三、準備：

    想要使用 Skype 視訊，必須準備以下事項。

    • 寬頻網路環境
    • Skype 用網路攝影機
    • 更新可支援 Skype 視訊之電視軟體。
    • 註冊一個Skype帳號或使用E-mail電子郵件信箱去申請一個新的 Skype 帳號。

    注意：根據您所使用的網路環境，視訊通話品質可能會有差異。

    四、購買網路攝影機：

    依據以下網頁的說明購買相容 Skype 之網路攝影機。 可利用 http://freetalk.me/product/sharp/
    注意：‧SHARP公司並無販賣 Skype FREETALK 網路攝影機。 ‧其他網路攝影機無法相容 SHARP AQUOS 液晶電視。

    五、更新您的電視：

    1. 按下遙控器上之藍色按鍵或是輸入( INPUT )鍵顯示"輸入"畫面。
    2. 確認在"輸入"畫面裏是否有 Skype 選項。
      假如有" Skyp e"：您不需要更新電視軟體，直接到下一頁“開始使用”。
      假如沒有" Skyp e"：請參閱 AQUOS 全球網站說明並依步驟3下載更新軟體到 USB 隨身碟上
      。 * AQUOS全球網站網址: http://www.aquos-world.com/index_en.html
    3. 更新可支援 Skype 視訊之電視軟體。
    4. 註冊一個Skype帳號或使用E-mail電子郵件信箱去申請一個新的 Skype 帳號。

    請依照您電視的使用說明書來更新軟體。
    • 液晶電視數位選台軟體更新
    • LC-42D63T / LC-46D63T
    • 2014-12-30
    • 規格表
    • 下載

    一、說明：

    本版本可接收數位電視原本的節目及新發射的HD台節目

    二. 適用此更新軟體之機型如下：

    LC-42D63T / LC-46D63T

    三. 數位軟體更新流程如下:

    1. 將軟體檔rootfs.bin放在USB隨身碟第一層，不要放在任何目錄之下。
    2. 將電視電源打開。
    3. 電視開機後,按下遙控器目錄選單，移動游標到最右邊的系統選項。
    4. 將USB隨身碟插入電視 USB 插槽。
    5. 再按下遙控器向下，移動游標到“軟體更新”，按下確認鍵。
    6. 更新完成後系統會顯示目前軟體版本。

    注意: 在軟體更新期間，請確保電力不中斷

    一、改版原因 :
